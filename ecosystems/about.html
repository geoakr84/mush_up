About

This website was made for the second assignment of the course Social Media Ecosystems which is a mandatory course of the master degree Social Media and Web Technologies. The web project that I developed is related to services that are provided by a variety of social media. The functions of theses services were implemented with the use of API that each social media has made available online.

The topic of my web project is about Vaxjo City and everything that is related to it. The site itself is a mush-up of data that regards information about the weather, employment, restaurants, news , etc. For the implementation of the web page I used mostly javascript, css and jquery.

The site can be expanded and enhanced with more features and services, in order to evolve in a complete tourist online guide. The users of the site would have the capability to interact with each other and with the site, to obtain the information that they seek.
